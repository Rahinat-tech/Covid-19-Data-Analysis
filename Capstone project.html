COVID 19 DATA ANALYSIS¶

1. INTRODUCTION:¶

The COVID-19 pandemic has profoundly impacted global health, economies, and societies. This analysis focuses on the “Covid_19_complete_clean.csv.zip” dataset to derive significant insights into the pandemics spread, mortality rates, and recovery trends across various countries. By employing advanced data analysis techniques, statistical methods, and machine learning models, we aim to uncover the key factors influencing COVID-19 outcomes and offer recommendations for future policy-making and research.

iv. FEAUTURE ENGINEERING

Daily Growth Rate by calculating the daily growth rate of confirmed cases.

Mortality Ratio Compute deaths as a proportion of confirmed cases.

Cases Per Population If population data is available, we will compute cases per 100,000 people.

In [34]:
# Daily growth rate
df['DailyGrowthRate'] = df.groupby(['Country/Region', 'Province/State'])['Confirmed'].pct_change().fillna(0)

# Mortality ratio
df['MortalityRatio'] = (df['Deaths'] / df['Confirmed']).replace([np.inf, -np.inf], 0).fillna(0)

# Cases per population
# Assuming population data is available as a separate dataset
population_data = {
    'usa': 331002651,  # Example: United States population
    'brazil': 212559417,  # Brazil population
    # Add other countries...
}
df['Population'] = df['Country/Region'].map(population_data)
df['CasesPer100k'] = (df['Confirmed'] / df['Population'] * 100000).fillna(0)

# Ensure index is a DatetimeIndex and set frequency
ts_data.index = pd.to_datetime(ts_data.index)  # Ensure it's a DatetimeIndex
ts_data = ts_data.asfreq('D')  # Explicitly set daily frequency

# Build and fit ARIMA model
from statsmodels.tsa.arima.model import ARIMA

model = ARIMA(ts_data, order=(1, 1, 1))
fit_model = model.fit()

# Forecast and plot
forecast = fit_model.forecast(steps=30)
future_dates = pd.date_range(start=ts_data.index[-1], periods=30, freq='D')
